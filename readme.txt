== Description ==

This plugin automatically add [Firebug Lite](http://getfirebug.com/lite.html) to your blog for the admin (> 7).

== Installation ==

1. Unzip the file firebug-lite.php or the folder /firebug-lite/
2. Upload it in /wp-content/plugins/ of your Wordpress installation
3. Activate the Plugin
4. And that's it!
